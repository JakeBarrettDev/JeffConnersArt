Jeff Conners explicitly asked for cyan and orange as the site's accent colors. These are defined as `--cyan` and `--orange` CSS variables. Do not swap them out for gold, amber, or other palettes. They appear in:
- `.accent` gradient text (headings)
- Active carousel dots
- Nav hover underlines (cyan)
- Button hover fills (orange)
- Commission panel hover border (orange)
- Category filter active state (cyan)
